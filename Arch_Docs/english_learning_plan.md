### 学习素材与细节  
**1. 音标与发音技巧**  
- **每日必学**：  
  - 上午：学习2-3个音标（如 /æ/, /ɛ/, /ɪ/），搭配10个单词（如 *cat* [kæt]、*bed* [bɛd]、*sit* [sɪt]）。  
  - 下午：通过音频按钮（[🔊]）跟读，重点练习爆破音（如 /p/, /b/）和元音弱化（如 /ɪ/ → /ə/）。  
- **发音技巧总结**：  
  - **连读**：如 *I am* → *I'm*（/aɪm/）。  
  - **弱读**：如 *to go* → *tə ɡəʊ*（/tə ɡəʊ/）。  
  - **重音规则**：名词重读（如 *book* [bʊk]），动词弱读（如 *to go* → *tə ɡəʊ*）。  

**2. 单词与短语**  
- **每日词汇**：  
  - 例：Day 1（上午）：*apple* [ˈæpəl]、*book* [bʊk]、*chair* [tʃɛr]。  
  - 例：Day 2（下午）：*water* [ˈwɔːtər]、*door* [dɔːr]、*window* [ˈwɪndəʊ]。  
- **短语练习**：  
  - 例：*What time is it?* [wʌt taɪm ɪz ɪt?]（带音频按钮）。  

**3. 听力材料**  
- **每日对话**：  
  - 例：Day 8（下午）：  
    - *A: Can I help you?* [kæn aɪ hɛlp juː?]  
    - *B: Yes, I want to buy a shirt.* [jɛz aɪ wɒnt tə baɪ ə ʃɜːrt]  
    - 配音频按钮（[🔊]）和听力填空练习。  

**4. 语音播放按钮**  
- 所有单词、短语和对话均标注音标，并在右侧添加[🔊]按钮（如：*hello* [hɛˈloʊ] [🔊]）。  

**5. 难点突破**  
- **每日总结**：  
  - 例：Day 10（下午）：  
    - 难点：元音 /æ/ vs. /ɑː/（如 *cat* [kæt] vs. *car* [kɑːr]）。  
    - 练习：通过对比发音和例词（如 *cat*, *car*, *carpet* [ˈkɑːrpɪt]）。

## 🎯 发音技巧总结（可随时复习）  
1. **连读**：  
   - 如 *I am* → *I'm* [aɪm]  
   - 如 *to go* → *tə ɡəʊ*  
2. **弱读**：  
   - 功能词（to, is, at）弱化为 /ə/  
   - 如 *I want to go* → *aɪ wɒnt tə ɡəʊ*  
3. **重音规则**：  
   - 名词重读（*book* [bʊk]）  
   - 动词弱读（*to go* → *tə ɡəʊ*）  
4. **爆破音**：  
   - /p/ 和 /b/ 在句尾爆破（如 *pat* [pæt]）  
   - /t/ 和 /d/ 在句尾爆破（如 *cat* [kæt]）
